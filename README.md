## INSTALLATION [Termux]

* `apt update`
* `apt upgrade`
* `pkg install python`
* `pkg install python2`
* `pkg install git`

* `pip3 install -r requirements.txt`
* `chmod +x *`
* `cd start`
* `bash start`
